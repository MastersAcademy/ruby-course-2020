        I think the most useful feature in this version is Hash#to_proc

        h = { a: 1, b: 2, c: 3, d: 4 }
        [ :a, :c, :d, :b ].map(&h)
        # => [ 1, 3, 4, 2 ]

        And your hash can have a default value or proc set for non entry values. So much win!

                          The parameter syntax given to map with the ampersand is a call to to_proc

                          class A
                            def A.to_proc
                              lambda {|v| "#{v}y" }
                            end
                          end

                          %w'a s d f'.map(&A)
                          ["ay", "sy", "dy", "fy"]
