Main steps in learning Ruby

Step 1: Installing Ruby

First of all you must installing Ruby on yours machine. I prefer RVM installation

RVM (“Ruby Version Manager”)

Step 2: Learn the basics of Ruby

There are many resources on the Internet to help you with this.

But still I will advise you some resources:

  • Ruby WikiBook
  • The Odin Project

Step 3: Read the documentation

Sooner or later you will get stuck in solving a certain problem.

But don't despair Ruby's documentation will help you solve your problems.

Take a look at the documentation more often and experiment with your code.

Step 4: Time Management

Many people ignore this point. But it is the key to studying any science as effectively as possible.

Time management allows you to find a balance between study and rest.

Step 5: Learn Git

Git is a distributed version-control system for tracking changes in any set of files.

Originally designed for coordinating work among programmers cooperating on source code during software development.

Thus, no matter what programming language you learn, knowledge of GIT is required.

And the sooner you start studying it the better.


I could describe more steps, but these are basic. The main thing is not to stop, keep coding :)
